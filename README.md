### 👨‍💻 About Me

I'm **Mohammad Musa**, a passionate and self-driven **Full Stack MERN & React Native Developer** from **Bangladesh** 🇧🇩.  
I love building **modern, scalable, and user-friendly web and mobile applications** using the latest technologies in the JavaScript ecosystem.

Currently, I’m working on **“Cholo – A Transport Management and Vehicle Tracking System”**,  
a project designed to modernize university transport management using **Node.js**, **Express**, **React Native**, **MongoDB**, and **Socket.IO** for **real-time vehicle tracking**.

I’m deeply interested in backend architectures, API design, and real-time communication systems.  
My goal is to craft clean, efficient, and maintainable solutions that make technology accessible and impactful.
